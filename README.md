# Installation 
```console
git clone https://github.com/Joint55internal/Stucus.git
```
Navigate to Stucus
```console
cd Stucus
```
Compile the source code:
```console
g++ stucus.cpp -o stucus
```
# Usage
To use Stucus, execute the following command:
```console
./stucus
```
